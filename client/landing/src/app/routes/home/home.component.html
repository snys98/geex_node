Modern architecture for developers&business friendly software solutions.

Geex Framework is a complete infrastructure to create modern web applications by following the best practices and conventions of web development.

  • 01Monolithic, with microservice support

    The core framework & pre-built modules are highly cohesive monoliths, but designed with microservice architecture in mind.

  • 02 Modular Architecture, which is extensible & replaceable

    Geex provides a module system that allows you to develop reusable application modules, to leverage all your valuable business logic.

  • 03 Domain Driven Design, with implicit best practice

    Geex always keeps DDD in mind, we bootstrap many DDD concepts in framework and you can use them without effort. Writing clean code is as natural as breathing the air.

  • What are the differences between the open source Geex Framework and Geex Commercial?

    Geex framework is a modular, themeable, microservice compatible application development framework for ASP.NET Core. It provides a complete architecture and a strong infrastructure to let you focus on your own business code rather than repeating yourself for every new project. It is based on the best practices of software development. Geex framework is completely free, open source and community-driven. It also provides a free theme and some pre-built modules (e.g. identity management and tenant management). Geex Commercial adds the following benefits on top of the Geex framework: Professional Bootstrap UI templates Enterprise ready, feature-rich, pre-built Application Modules (e.g. Identity Server management, SaaS management, language management) Tooling to support your development productivity (e.g. Geex Schematics) Training & Support Visit the following link for more information https://todo.com
